# Source Priority

Use this order unless the user specifies a tighter lane:

1. User-provided materials
2. `document_store` knowledge-base or competitive-doc sources
3. Recent `internal_messaging` field evidence when the user asks for internal context
4. Official competitor and partner sources
5. Investor or regulatory materials when relevant
6. public research or user-provided market intelligence when relevant
7. Secondary coverage

Keep source provenance explicit. Do not collapse different sources into one generic claim.
