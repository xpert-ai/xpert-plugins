# Output Contract

Default sections:

1. Context
2. Seller Baseline
3. Competitive Landscape Summary
4. Competitor Snapshot
5. Comparison Matrix Takeaways
6. Response Strategy
7. What Not To Overclaim
8. Sources

For a very short `brief`, these can be compressed to:

1. Competitor Snapshot
2. What They Are Claiming
3. What Is Confirmed
4. Why Buyers Might Care
5. Weak Spots and Open Questions
6. Response Strategy
7. What Not To Overclaim
8. Sources

If `mode=objections`, focus the output on objection handling and proof points.

If `mode=account_overlay`, add one section called `Account Implications` and clearly label any directional inference.

Use placeholder examples when showing structure:

- seller: `MyCompany`
- competitor: `Competitor Alpha`
- customer/account: `ExampleCorp`
