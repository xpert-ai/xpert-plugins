# Public Equity Investing User Context Config

Plugin authors edit this file when adding, removing, renaming, or clarifying the starter categories copied into a user's `user-context.md`. The initializer adds `## Saved Links And Context` with `status: not provided` beneath each category.

## Default Categories

# Investor Profile And Output Style

- Description: Audience mode, mandate style, preferred memo or report depth, and PM or research-team conventions that should shape future Public Equity Investing work.

# Portfolio Watchlist And Repository Pointers

- Description: Watchlist locations, research repositories, approved trackers, and recurring review sources for public-equity workflows.

# Research Source Priorities

- Description: Preferred first-look providers, filing and transcript conventions, trusted internal research locations, and source-freshness rules.

# Model Valuation And Estimate Conventions

- Description: Model templates, valuation methods, estimate-source preferences, KPI definitions, and sector-specific conventions.

# Thesis Catalyst And Risk Tracking Conventions

- Description: Thesis-tracker locations, catalyst calendars, monitoring cadence, risk thresholds, and review triggers.

# Compliance Circulation And Review Rules

- Description: Distribution boundaries, citation expectations, compliance review, and client-facing or publication gates.
