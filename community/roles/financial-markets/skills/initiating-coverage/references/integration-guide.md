# Integration Guide

## Upstream skills

### financial-source-of-truth

Use first when sources are messy, conflicting, stale, or mixed between user files, filings, provider data, and web. It supplies evidence hierarchy and citation protocol.

### company-tearsheet

Use to create a concise baseline profile before a full initiation, especially when the user provides little context.

### financials-normalizer

Use when historical financials are embedded in filings, PDFs, spreadsheets, data-room exports, or messy tables.

### excel-data-cleaner

Use when the source workbook has messy tabs, headers, units, dates, hidden rows, broken formats, or inconsistent mappings.

### equity-model-update

Use to update or build an operating model from latest filings, earnings releases, transcripts, KPIs, and consensus estimates.

### earnings-preview / earnings-deep-dive

Use when the initiation depends on a recent or upcoming earnings event.

## Parallel or downstream skills

### Valuation skills

Use for detailed DCF, comps, SOTP, valuation range, and target price support when valuation is the core work product.

### scenario-sensitivity-generator

Use for bull/base/bear, price target sensitivities, downside case, and break-even assumptions.

### long-short-pitch

Use after initiation when the user wants a trade expression: position, catalyst, hedge, risk/reward, and kill signals.

### thesis-tracker

Use after initiation to track KPIs, catalysts, estimate revisions, confirm/disconfirm signals, and thesis status.

### portfolio-risk-management

Use when a thesis needs pair trade, factor hedge, proxy hedge, options hedge, or exposure management.

### memo-builder

Use to convert the initiation into an IC memo, PM memo, committee note, or client-ready summary.

### deck-report-qc / style-guide-adapter

Use before publishing or sharing externally to align formatting, sources, disclosures, and style.

## Handoff fields

When handing off to another skill, provide:
- company/ticker,
- report mode,
- thesis pillars,
- key debates,
- model drivers,
- valuation method and target price,
- catalysts,
- risks and disconfirming signals,
- source register,
- assumptions register,
- stale-data log,
- open questions.
