# Valuation Rules

Deferred reference for `sector-context-overlay` insurance sector lens. Load only when the task needs valuation, public-equity security context and Credit Markets handoff, or Credit Markets handoff rules.

## Use insurance-specific valuation, not generic corporate valuation

## Equity valuation hierarchy

### P&C / specialty / reinsurance
Use these lenses first:
1. price / tangible book or clean book
2. price / earnings on normalized EPS
3. normalized ROE relative to cost of equity
4. underwriting franchise value plus investment leverage

### Life / retirement / annuity
Use these lenses first:
1. price / adjusted book value or clean book
2. price / adjusted operating EPS or distributable EPS
3. value of remittances or free cash generation
4. appraisal value or embedded value where relevant and credible

### Valuation rules
- Higher multiples are justified by reserve quality, underwriting moat, capital-light growth, rating resilience, cleaner asset books, and better cash-remittance visibility.
- Two insurers with similar current EPS can deserve radically different multiples if one is under-reserved, overexposed to asset risk, or capital constrained.
- For life insurers, do not confuse mark-driven book volatility with true franchise impairment, but do not ignore capital and liquidity consequences either.
- Use sum-of-the-parts only when segments truly have different capital intensity and investors can realistically access that value.

## Debt valuation hierarchy
- spread versus expected loss and recovery
- distance to rating or solvency impairment
- holdco liquidity and remittance capacity
- structural subordination to policyholders and opcos
- catastrophe, reserve, and asset-spread stress path
