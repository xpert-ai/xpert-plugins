# Research Output Overlay Contract

Use this PM handoff block when sector context is injected into another Public Equity Investing skill.

Required fields:

- `sector_overlay.selected_lens`
- `issuer_archetype`
- `mandate_lens`
- `pm_debate`
- `kpi_set`
- `valuation_implications`
- `dashboard_modules_to_add`
- `source_gaps`
- `red_flags`
- `secondary_lenses`

The owning skill keeps the artifact. This overlay supplies sector-specific judgment, KPI hierarchy, valuation conventions, red flags, and missing evidence.
