# Trade Compliance Workbench V2 Prototype

这是新版外贸合规工作台的静态界面原型，和现有插件实现隔离。

## 本地预览

直接打开 `index.html` 即可查看。

也可以启动静态服务：

```bash
pnpm start
```

默认地址：

```text
http://localhost:5177
```

## 页面结构

- 待处理
- 管控规则库
- 供应商商品
- 销售发票
- 工具 / HS 编码查询

## 设计原则

- 管控商品是规则库
- 供应商商品是业务实例
- HS 编码查询是辅助工具
- 销售发票是最终输出
- AI 解析结果必须经过人工确认才进入正式数据
