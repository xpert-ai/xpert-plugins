Add server-side XLSX automation with bounded workbook reads, typed value and formula edits, sheet management, immutable file versions, optimistic concurrency, idempotent retries, version restoration, and downloadable Excel artifacts.

Retain imported XLSX files in scoped workspace storage, synchronize successful edits back to the Univer Workbench, and expose current-file downloads and automation status in the Office Editor UI.
