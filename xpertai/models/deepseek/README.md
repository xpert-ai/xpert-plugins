# @xpert-ai/plugin-deepseek

DeepSeek AI model provider plugin for XpertAI platform.

## Features

- Support for DeepSeek Chat models
- Support for DeepSeek Reasoner (R1) with advanced reasoning capabilities
- OpenAI-compatible API
- Streaming support
- Tool calling support

## Supported Models

- `deepseek-chat` - General purpose chat model with 64K context
- `deepseek-reasoner` - Advanced reasoning model (R1) with enhanced problem-solving

## Installation

This plugin is part of the XpertAI plugins monorepo.

## Configuration

Get your API key from [DeepSeek Platform](https://platform.deepseek.com/api_keys).

### Required Credentials

- `api_key`: Your DeepSeek API key
- `base_url` (optional): Custom API endpoint, defaults to `https://api.deepseek.com/v1`

## Usage

The plugin will be automatically loaded by the XpertAI platform when installed.
