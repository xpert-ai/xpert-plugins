Template preview images for the Sites workbench.

Put images in this directory using one of these file names. The first existing file in each row is used.

- Onboarding Hub: `sites-onboarding-hub.png`, or `onboarding-hub.jpg/jpeg/png/webp`
- Enablement Hub: `sites-enablement-hub.png`, or `enablement-hub.jpg/jpeg/png/webp`
- Pulse Dashboard: `sites-pulse-dashboard.png`, or `pulse-dashboard.jpg/jpeg/png/webp`
- Sparkboard: `sites-idea-intake.jpg`, or `sparkboard.jpg/jpeg/png/webp`
- Launch Cal: `sites-launch-cal.jpg`, or `launch-cal.jpg/jpeg/png/webp`
- Event Planning Hub: `sites-event-planning-hub.jpg`, or `event-planning-hub.jpg/jpeg/png/webp`

The build copies this folder into `dist`, and the view provider embeds existing images into the remote component HTML as data URLs.
